## C. VISUAL DESIGN DIRECTION

**Color Palette:**
- Navy `#0D1B2A` — authority, trust, premium
- Teal `#00B4D8` — innovation, digital, modern
- Gold `#F4A261` — warmth, standout moments, milestones
- Off-white `#F8FAFC` — clean background sections
- Success green `#10B981` — availability badge, positive signals

**Typography:**
- Headings: `Playfair Display` (serif) — premium, confident
- Body: `Inter` (sans-serif) — modern, highly readable, ATS-neutral

**Layout Style:**
- Single-page scroll, section-by-section reveal
- Max-width 1100px container, generous whitespace
- Grid-based layouts (2-col, 3-col, 4-col responsive)
- Left-aligned text for readability (European recruiter norm)

**UI Components Used:**
- Animated counter cards (achievement numbers)
- Timeline with alternating left-right entries (desktop) / linear (mobile)
- Skill chip tags (grouped by domain)
- Card components with hover states
- Badge/pill components for status signals
- Fade-in on scroll (IntersectionObserver)
- Fixed nav with hamburger mobile menu
- Scroll-to-top button

**Number Treatment:**
- Large, gradient-clipped numerals in the achievements section
- Animated counter effect on scroll entry
- Source attribution beneath each number for credibility
